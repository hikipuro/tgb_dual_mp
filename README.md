# TGB Dual MP
Multi Platform Gameboy Emulator

# How to build

```bash
$ npm i
$ npm run build
```

# How to run

```bash
$ npm run app
```

# Default key config

|GB button|Keyboard|
|---------|--------|
|A        |X       |
|B        |Z       |
|Start    |Enter   |
|Select   |Shift   |
|Rapid    |Tab     |
|Repeat   |A       |
|Pause    |P       |

# Features

- Any PC OS supported (Windows, Mac, Linux)
- Gameboy, Gameboy color emulation
- Realtime save state
- Gamepad support (Web Gamepad API)
